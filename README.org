
Методические указания студентам к выполнению курсовой работы по
дисциплине "Стратегии коммерциализации инноваций"

Эту методичку сделать простой адаптацией методичек других институтов.

Работы:
- разбить по главам
- переписать весь материал
- составить список тем
- собрать в pdf, doc

Структура файлов:
- README.org
- Makefile
- main.org :: главный сборочный файл с конфигурацией и титульный лист
  - Титульный лист
    - Организация
    - Кафедра
    - Дисциплина
    - Наименование издания
    - Автор
    - Версия
    - Дата
- intro.org
- requirements.org :: Основные требования к курсовой работе
- writing.org :: Этапы написания курсовой работы
- biblio.org
- theme-selection.org
- themes.org

План методички:
- Титульный лист
- Введение
- Основные требования к курсовой работе
- Этапы написания курсовой работы
- Написание текста курсовой работы
- Подбор и обработка материалов
- Правила подготовки и оформления
- Защита
- Правила оформления списка литературы
- Критерии оценки
